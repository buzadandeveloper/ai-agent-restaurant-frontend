# AI Restaurant

Frontend application for an AI Restaurant

---

### - Prerequisites

Before you begin, ensure you have the following installed:
- **Node.js** (v18 or higher recommended)
- **pnpm** (v8 or higher)

If you don't have pnpm installed, you can install it globally:
```bash
npm install -g pnpm
```

### - Installation

Clone the repository:
```bash
git clone https://github.com/buzadandeveloper/ai-agent-restaurant-widget-frontend.git
```

2. **Install dependencies:**
```bash
pnpm install
```

4. **Start the development server:**
```bash
pnpm dev
```

This will run both the Vite dev server and TypeScript compiler in watch mode concurrently.

5. **Open your browser:**

Navigate to `http://localhost:5173`
